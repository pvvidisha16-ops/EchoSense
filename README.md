# EchoSense

## Overview

EchoSense is a simple assistive web application designed to help paralysed or mobility-impaired patients communicate urgent needs to caregivers using voice commands. The patient can speak keywords like **“help”** or **“pain”**, and the system instantly alerts the caregiver through a monitoring dashboard.

## Features

* **Voice Detection:** Uses speech recognition to detect distress keywords.
* **Real-Time Alerts:** Caregivers receive alerts with a message and timestamp.
* **Caregiver Dashboard:** Displays live alerts and allows monitoring of patient requests.
* **Patient Details:** Shows basic patient information for reference.
* **Alert History:** Stores recent alerts for review.
* **Medication Reminder:** Provides simple reminders for medication.

## Tech Stack

* **Frontend:** HTML, CSS, JavaScript
* **Backend:** Python (Flask)
* **APIs:** Web Speech API for speech recognition

## Purpose

EchoSense aims to improve accessibility and safety for paralysed patients by enabling quick voice-based communication with caregivers.
